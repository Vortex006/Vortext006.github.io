什么是BootStrap？

Bootstrap是一个用于快速开发Web应用程序和网站的前端框架。

在学习Bootstrap前，必须具备HTML、CSS、JavaScript的基础知识，只需要了解相关的class、标签名称所代表的含义，在构建页面时，导入Bootstrap的JS、CSS等，它就会表现出相应的效果。

---参考《Bootstrap基础教程》p1
BootStrap的优点
使用BootStrap 不使用BootStrap
HTHL

摒弃了复杂而又毫无意义的标签，引入全新的标签，使网页的语义性大大增加。

标签复杂而又臃肿，不够灵活
CSS

使用Sass创建的CSS，是动态的CSS，所需代码更少，解析更容易，阅读更轻松

书写繁琐、数值固定不变修改、频繁调试
JavaScript

使用jQuery库，内置多种JS函数，需要的功能只需要调用相应的函数，十分方便

JS语法书写复杂、代码量多且不易于复用
移动设备

移动设备优先、响应式布局，对于电脑、手机、平板等设备都能够自适应

需要根据不同的设备重构页面、费时费力
通用性

IE、Firefox、Google、Edge等主流浏览器都支持BootStrap

需要针对不同浏览器做适配、不利于开发
---参考《Bootstrap基础教程》p2
如何使用BootStrap？

用户可以前往BootStrap中文网直接下载BootStrap最新版本即可。

下载完毕后，直接在HTML文件中引入BootStrap的CSS和JS文件：

<link rel="stylesheet" href="bootstrap.min.css">
<script src="jquery-3.4.1.js"></script>
<script src="bootstrap.bundle.js"></script>

注：需要优先引入jquery文件
